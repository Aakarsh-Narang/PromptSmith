# 🌟 PromptSmith — AI-Powered Code Generation Tool

**PromptSmith** is an AI-powered code generation tool that bridges the gap between natural language and production-ready UI code.  
It allows developers to generate **HTML**, **React**, or **JavaScript** components instantly from simple descriptive prompts.

> 🚀 **Turn natural language prompts into production ready UI code instantly.**  
> Built as a showcase project for developer tools and AI-first development workflows.

---

## ✨ Key Features

- 🤖 **AI-Driven Code Generation** — Generate HTML, React (JSX), or vanilla JavaScript code effortlessly  
- 📱 **Responsive & Modern** — All generated components follow modern UI/UX and responsive design principles  
- 💾 **Smart History** — Save prompts and generated code automatically for later reference  
- ⚡ **Real-Time Preview** — Instantly visualize UI components as they are generated  
- 🎨 **Tailwind CSS Styling** — Utility-first, consistent, and easily customizable  
- 🛠️ **Extensible Output** — Easily extend supported output languages via simple backend changes  

---

## 🧠 Technologies Overview

PromptSmith seamlessly integrates **Google AI** and modern web development tools.

| **Component**     | **Technology**       | **Purpose**                                        |
|--------------------|----------------------|----------------------------------------------------|
| **AI Engine**      | Google Gemini        | Converts natural language into code                |
| **AI Workflow**    | Firebase Genkit      | Handles prompt workflows and AI orchestration      |
| **Data Storage**   | Firebase Firestore   | Saves prompts, code history, and user interactions |
| **Authentication** | Firebase Auth        | Manages user sessions and authentication           |
| **Hosting**        | Vercel / Firebase Hosting | Deploys the app at scale                         |
