# 🌟 PromptSmith — AI-Powered Code Generation Tool

**PromptSmith** is an AI-powered code generation tool that bridges the gap between natural language and production-ready UI code. It allows developers to generate **HTML**, **React**, or **JavaScript** components instantly from simple descriptive prompts.
**Turn natural language prompts into production-ready UI code instantly**

PromptSmith is a cutting-edge web application that leverages AI to generate clean, semantic, and responsive UI code from simple natural language prompts. Whether you're learning frontend development, prototyping an MVP, or building real-world applications, PromptSmith bridges the gap between imagination and implementation.

> 🚀 **Built as a showcase project for Google Developer tools and AI-first development workflows.**

---

## ✨ Key Features

🤖 **AI-Driven Code Generation** — Generate HTML, React (JSX), or vanilla JavaScript code effortlessly  
📱 **Responsive & Modern** — All generated components follow modern UI/UX and responsive design principles  
💾 **Smart History** — Save prompts and generated code automatically for later reference  
⚡ **Real-Time Preview** — Instantly visualize UI components as they are generated  
🎨 **Tailwind CSS Styling** — Utility-first, consistent, and easily customizable  
🛠️ **Extensible Output** — Easily extend supported output languages via simple backend changes

## 🧠 Technologies:

PromptSmith seamlessly integrates Google AI and development tools:

- Generate clean, semantic, and responsive code using **Tailwind CSS**.
- Supports multiple output formats: HTML, React (JSX), JavaScript (DOM).
- Easily extendable to support additional output languages via simple backend changes.
- Provides a seamless developer experience for MVPs, prototypes, or learning frontend development.
- Voice prompt support for faster input.
| Component | Google Technology | Purpose |
|-----------|-----------------|---------|
| **AI Engine** | Google Gemini | Converts natural language into code |
| **AI Workflow** | Firebase Genkit | Handles prompt workflows and AI orchestration |
| **Data Storage** | Firebase Firestore | Saves prompts, code history, and user interactions |
| **Authentication** | Firebase Auth | Optional user management and sessions |
| **Hosting** | Vercel / Firebase Hosting | Deploys the app at scale |

## 💻 Tech Stack:

### Frontend
- **Framework**: Next.js 14 with App Router  
- **Language**: TypeScript for type safety  
- **Styling**: Tailwind CSS + ShadCN/UI components  
- **State Management**: React hooks and Context API  

- **Frontend:** Next.js, Tailwind CSS
- **Backend/AI:** Google Gemini, Firebase Genkit
- **UI System:** CampEdUI (ShadCN)
- **Version Control:** Git & GitHub
### Backend & AI
- **AI Model**: Google Gemini 2.0  
- **AI Framework**: Firebase Genkit  
- **Database**: Firebase Firestore  
- **Authentication**: Firebase Auth  

### Development & Deployment
- **Package Manager**: npm / yarn  
- **Build Tool**: Next.js bundler  
- **Deployment**: Vercel + Firebase Hosting  

---

### Prerequisites
- Node.js 18+ and npm/yarn  
- Firebase project with Firestore enabled  
- Google AI key for Gemini API access
